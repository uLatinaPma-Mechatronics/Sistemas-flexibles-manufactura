RESUMEN:
Sistema Flexible que hace 2 paletizados diferentes de acuerdo a las dimensiones de las cajas, proporcionando 2 pallets con cajas de un solo tipo, es decir las cajas pequeñas con las pequeñas y las grandes con las grandes, además este proyecto se le adapta un sistema de seguridad que hace que el robot deje de trabajar al haber sensado algo.
